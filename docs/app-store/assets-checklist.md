# App Store Asset Checklist

## Required Assets

- App icon: 1024x1024 PNG, no transparency
- iPhone screenshots from current production build
- iPad screenshots if tablet support remains enabled

## Screenshot Capture Set

- iPhone 6.9" display class
- iPhone 6.5" display class
- iPad 13" display class if shipping iPad

## Suggested Screenshot Sequence

1. Home/dashboard
2. Meal scan result
3. Whole-food product analysis
4. AI chat
5. Recipe browse or meal planning

## Before Upload

- Remove placeholder or internal-only data.
- Use production-safe copy and imagery.
- Ensure camera/photo permission prompts are not visible unless intentionally captured.
