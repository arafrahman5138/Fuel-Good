# Fuel Good Support

For product support, account issues, bug reports, or TestFlight feedback:

- Email: support@fuelgood.com
- Response target: within 2 business days

## Include in Your Message

- Device model
- iOS version
- App version/build
- What you were doing when the issue happened
- Any request ID or error text shown in the app

## Common Topics

- Sign-in issues
- Scan upload failures
- Missing or incorrect meal analysis
- Account and privacy requests

Before launch, publish this page at the production support URL.
